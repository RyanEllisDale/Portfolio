    About Astrobot New Worlds

    Astrobot: New Worlds is a vertical slice of a potential sequel to Astro Bot, developed as part of Teesside University’s 2025 Journeyman project. Set atop vibrant, synthwave-inspired skyline rooftops, the game is a cooperative third-person platformer focused on creative traversal and puzzle-solving. As Programming Lead, I was responsible for directing the technical implementation and building core gameplay systems using Unreal Engine 5’s Blueprint system. My primary contribution was designing and developing a charging mechanic, allowing players to power and overcharge in-world electronics to progress through the environment. This was a cross-discipline collaboration between programmers, artists, and designers. We coordinated using GitHub for version control and Miro for planning and design, fostering a professional and iterative development process that mirrored real industry workflows.

    What I learnt:

    Working on Astrobot: New Worlds taught me a ton—not just about using Unreal Engine, but about working as part of a proper development team. I got really hands-on with Blueprints and learned how to build more complex systems like the charging mechanic, which was a fun challenge. Like a lot of students who’ve done projects like this, I found that working with artists and designers pushed me to improve how I communicate and manage my time. Using tools like GitHub and Miro helped keep everything organized and gave the whole thing a more professional feel. Overall, it was a great experience that gave me a better idea of what it’s like to work in the games industry.

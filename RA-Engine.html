    About RA-Engine

    The Retro Adventure Engine is a lightweight 2D engine designed for creating retro-style top-down RPGs, inspired by classics like The Legend of Zelda and Pokémon. Built in C++, it utilizes the SFML and ImGUI libraries for graphics and interface rendering. Currently, the engine supports sprite rendering from sprite sheets, the creation and movement of character entities, and basic game functionality.The engine’s core appeal lies in its commitment to a retro aesthetic. To reinforce this, intentional limitations are built into the design—such as a maximum sprite size of 64x64 pixels, reduced colour depth, and the absence of modern transparency (e.g., PNG alpha channels), instead relying on user-defined alpha colours. Additionally, audio assets will be compressed to mimic the hardware constraints of older systems, further enhancing the nostalgic feel.

    What I learnt:

    Developing the Retro Adventure Engine provided valuable experience in low-level game engine architecture and C++ programming. It deepened my understanding of 2D rendering using SFML, GUI creation with ImGUI, and memory management practices. I also gained insight into designing systems with artistic and technical constraints in mind, such as sprite limitations and audio compression, to replicate retro hardware. This project reinforced key programming concepts like entity systems, input handling, and the balance between usability and authenticity in engine design.
